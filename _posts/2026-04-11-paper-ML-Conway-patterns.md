# [论文阅读] 基于 Conway 图案的机器学习驱动红外超表面频谱选择性设计

> - **标题**：Machine learning design of spectral-selective infrared metasurfaces based on Conway patterns
> - **期刊**：Physical Chemistry Chemical Physics (PCCP)，2025，27，17047–17054
> - **DOI**：https://doi.org/10.1039/d5cp01823d
> - **作者**：Chengyu Xiao‡, Jing Li‡, Wansu Hua‡, Yifan Zhang, Han Zhou\*（上海交通大学材料科学与工程学院 / 张江高等研究院未来材料创新中心）
> - **通讯作者**：Han Zhou（hanzhou_81@sjtu.edu.cn）
> - **发表时间**：2025-05-15

---

## 一、研究背景与问题提出

### 1.1 应用背景

红外超表面（metasurface）能够对热辐射的光谱、方向和偏振特性进行精确调控，在以下领域具有重要应用：

| 应用领域 | 关键功能 |
|----------|----------|
| 辐射制冷 | 在大气透明窗口（5–8 μm 和 8–13 μm）选择性发射热辐射 |
| 红外隐身 | 在特定波段隐藏热特征，规避探测 |
| 太空热管理 | 调控航天器散热通道 |
| 传感探测 | 多光谱特征识别 |

机器学习（ML）的引入显著加速了红外超表面的逆向设计，但现有方法存在两大关键瓶颈：

1. **结构设计空间受限**：现有方法多依赖简单几何图元（正方形、十字形等），限制了光谱调制的复杂度和精度
2. **材料选择局限**：材料库过小、缺乏系统性描述符，难以进行全面的材料-结构协同优化

### 1.2 现有方案不足

- **随机像素图案**：通过增加像素数量引入复杂性，缺乏内在结构组织和系统生成原则，导致冗余设计空间
- **固定材料搭配**：往往仅考虑一种或少数预定义材料，无法充分挖掘材料多样性带来的性能潜力

### 1.3 核心创新思路

本文提出用**Conway 生命游戏（Conway's Game of Life）细胞自动机**生成超表面图案，并结合 71 种材料数据库与深度神经网络/粒子群优化，构建一套完整的机器学习驱动设计平台。

## 二、核心创新点

| # | 创新点 | 具体内容 |
|---|--------|----------|
| 1 | **Conway 图案生成范式** | 首次将 Conway 生命游戏规则引入超表面图案生成，生成具有涌现复杂性（emergent complexity）和内在组织结构的拟态图案，取代简单几何图元 |
| 2 | **大规模多材料数据库** | 构建包含 71 种候选材料（21 种金属 + 34 种无机物 + 16 种聚合物）的数据库，配合 one-hot 编码，支持材料-结构协同优化 |
| 3 | **多层次特征融合神经网络** | 设计分级特征融合架构（staged feature fusion），将材料、图案（自编码器压缩至 256 维）、几何三类描述符渐进融合，避免高维信息冗余 |
| 4 | **自编码器图案压缩** | 对 64×64 像素图案矩阵（4096 维）采用自编码器（AE）压缩至 256 维潜在空间，保留关键结构信息同时大幅降低输入维度 |
| 5 | **图像滤波 + 对称性处理后处理** | 针对 Conway 图案边缘锐利导致的寄生共振模式和制备困难，引入图像滤波（平滑）和对称处理，提升可制造性并降低偏振依赖 |
| 6 | **一对多设计空间探索** | PSO 全局优化可从同一目标光谱找到多个结构不同的候选方案（"一对多"映射），传统逆向设计无法实现 |
| 7 | **双功能验证（辐射制冷 + 隐身）** | 在 5–8 μm 大气透明窗口同时实现高吸收率（隐身）和辐射制冷，两功能协同而非相互制约 |

## 三、研究方法详解

![image-20260411134458616](/assets/posts_figs/2026-04-11-paper-ML-Conway-patterns/image-20260411134458616.png)

### 3.1 超表面模型结构

采用**金属-绝缘体-金属（MIM，Metal-Insulator-Metal）三层结构**：

```
┌─────────────────────────────────────┐
│  顶层（mat_1）：Conway 图案金属层    │  ← 金属/无机物（55 种候选），提供带选择性调制
│  厚度 t1，64×64 像素网格，像素边长 a │
├─────────────────────────────────────┤
│  中间层（mat_2）：介质吸收层         │  ← 金属 + 聚合物（加入高分子红外吸收），增强 5–8 μm 吸收
│  厚度 t2                             │
├─────────────────────────────────────┤
│  底层（mat_3）：铝（Al）反射底板     │  ← 固定为铝，阻止透射，T ≈ 0
│  厚度 t3                             │
└─────────────────────────────────────┘
```

**周期公式**：$p = 64 \times a + 2 \times d$，其中 $d$ 为相邻单元间距

### 3.2 Conway 图案生成

基于 Conway 生命游戏的三条演化规则在 64×64 二维矩阵上迭代 50 次生成图案：

| 规则 | 描述 |
|------|------|
| **稳定存活** | 存活细胞周围有 2 或 3 个存活邻居 → 下一代继续存活 |
| **死亡** | 存活细胞周围邻居 < 2（孤立）或 > 3（拥挤）→ 死亡 |
| **诞生** | 死亡细胞周围恰好有 3 个存活邻居 → 下一代诞生 |

- 初始种子结构由随机数函数生成
- 迭代 50 代后形成具有内在组织的拟态复杂图案
- 共生成 **6000 种**可用图案库（作为设计空间的结构分量）

> **优势对比**：与纯随机像素图案相比，Conway 图案在相同填充率和材料下展现出更高的光谱选择性和发射强度，寄生峰更少。

### 3.3 描述符体系

本文构建了三类描述符的综合体系（图 2）：

![image-20260411134607564](/assets/posts_figs/2026-04-11-paper-ML-Conway-patterns/image-20260411134607564.png)

#### (a) 材料描述符
- 71 种候选材料的复折射率 $n = n_r + ik$
- **one-hot 编码**：每种材料表示为 71 维二值向量（对应材料位置为 1，其余为 0）
- 金属：索引 1–22；无机物：索引 23–56；聚合物：索引 57–71

#### (b) 图案描述符（图像描述符）
- 64×64 像素矩阵（4096 维）→ **自编码器（AE）** 非线性压缩 → **256 维潜在空间**
- 编码-解码结构保留拓扑和形状等关键结构信息，有效降低计算复杂度

#### (c) 几何描述符
- 6 个几何参数：三层厚度（$t_1, t_2, t_3$）、周期 $p$、像素边长 $a$、间距 $d$

### 3.4 正向预测网络（Forward Prediction DNN）

采用**分级特征融合（staged feature fusion）**架构：

```
几何描述符（6维）
    ↓ 全连接层（80神经元）→ 特征提取
    ↓ 与图案描述符（256维）拼接（concat）
    ↓ 隐藏层（400神经元）
    ↓ 与材料描述符（142维 = 71×2层）融合
    ↓ 隐藏层（400神经元）
    ↓
输出：200个波长点的光谱吸收率（覆盖 3–25 μm）
```

- 训练/测试集比例：**80:20**
- 验证集损失最终降至 **3.386 × 10⁻³**
- 正向预测精度高，与 FDTD 模拟结果高度吻合

### 3.5 逆向设计（PSO 全局优化）

**粒子群优化（Particle Swarm Optimization, PSO）** 与预训练正向网络结合：

1. 在定义的参数空间内随机采样初始种群
2. 训练好的 DNN 替代耗时的 FDTD 仿真，加速适应度评估
3. 以目标光谱（5–8 μm 高吸收 + 其他波段高反射）为优化目标
4. PSO 迭代找到最优参数组合

> **效率提升**：DNN 替代 FDTD 后，逆向优化速度大幅提升（FDTD 单次计算数小时 → DNN 推理毫秒级）。

### 3.6 后处理：图像滤波 + 对称处理

Conway 图案因像素化带来**锐利边缘** → 引入寄生共振模式，且不利于制备：
- **图像滤波**：平滑图案边缘，消除高频伪特征，优化红外响应
- **对称处理**：使图案具有对称性 → 降低偏振依赖性，入射光偏振方向改变时光谱响应稳定

### 3.7 数值仿真验证

使用 **FDTD（时域有限差分）** 仿真：
- 验证 PSO 优化后的超表面模型光谱响应
- 分析 x-y 平面（图案层和介质层）及 x-z 截面的电场分布
- 证明局域表面等离激元共振（LSPR）机制与介质层电损耗的协同作用

## 四、主要结果

![image-20260411144524012](/assets/posts_figs/2026-04-11-paper-ML-Conway-patterns/image-20260411144524012.png)

### 4.1 正向预测模型性能

- 验证集 Loss = **3.386 × 10⁻³**（训练充分，泛化良好）
- 预测光谱与 FDTD 真实值高度吻合（ESI 补充材料中有大量对比图）

### 4.2 PSO 逆向设计结果（图 3c–f）

优化生成了 4 个具有代表性的多功能超表面模型（Table S2），均实现了目标光谱特性：

- **代表案例（图 3c（上图c），Ni+PEI的蝴蝶形 Conway 图案）**：
  - 5–8 μm 波段吸收率：**0.83**
  - 其他波段反射率：**0.9**
  - 同时满足辐射制冷与红外隐身双功能需求

- 与相同填充率的随机图案相比：Conway 图案光谱选择性更高，寄生峰更少
- 与传统几何图案相比：整体吸收性能更优，选择比（selectivity ratio）更高

### 4.3 电场分析（图 4）

![image-20260411144623613](/assets/posts_figs/2026-04-11-paper-ML-Conway-patterns/image-20260411144623613.png)

以蝴蝶形模型为例：

| 波长 | 图案层（x-y 面） | 中间层（x-y 面） | 截面（x-z 面） |
|------|----------|----------|------|
| 3.6 μm | 弱场，非目标波段 | 弱场 | 场连续穿透 |
| 5.8 μm | **图案边缘强 LSPR**，中心耦合增强 | **目标波段强电场集中** | 场在图案层分裂衰减 |
| 7.5 μm | 同上，LSPR 特征明显 | 强电场 | 截面共振确认 |
| 16.7 μm | 场减弱，超出目标波段 | 弱场 | 场基本直接透过 |

**关键机制**：
1. 顶层 Conway 图案：LSPR 在 5–8 μm 提供带选择性吸收入口
2. 中间介质层（聚醚酰亚胺 PEI）：化学键固有吸收（C-O-C、C=O 等）在 5–8 μm 高效耗散电磁波能量
3. 底层铝板：完全反射，阻止能量透出（T ≈ 0）

**三层协同作用**：顶层选频 → 中间层耗散 → 底层反射，实现高效带选择性吸收。

## 五、主要结论

1. **Conway 图案优越性**：基于 Conway 生命游戏的图案生成方法比随机像素图案具有更好的光谱选择性和更少的寄生峰，说明**有序/规则的结构复杂性优于纯随机复杂性**

2. **大规模材料筛选可行性**：71 种材料 + one-hot 编码方案成功实现了机器学习框架下的多材料协同优化，显著拓展了设计空间

3. **分级特征融合有效性**：与直接高维参数输入相比，分级融合策略有效缓解了信息冗余，提升了各设计参数对性能预测的贡献平衡

4. **双功能验证**：成功在 5–8 μm 大气窗口同时实现辐射制冷所需的高吸收率（0.83）和红外隐身所需的高反射率（0.9 @ 其他波段）

5. **图像后处理价值**：滤波平滑和对称处理在不显著牺牲光谱性能的前提下，明显提升了图案的可制备性和偏振不敏感性

6. **"一对多"设计能力**：机器学习方法自然产生了单一目标光谱对应多种结构候选的映射，为实验验证提供了丰富的备选方案

## 六、研究方法框架总结

```
【数据生成阶段】
Conway 生命游戏（6000 种图案）
× 71 种材料（one-hot）
× 几何参数扫描
    ↓ FDTD 仿真
  30000+ 数据集（3–25 μm 光谱）

【机器学习训练阶段】
  自编码器（AE）压缩图案 → 256维
  分级特征融合 DNN 训练
    ↓ 验证 Loss ≈ 3.4×10⁻³

【逆向设计阶段】
  PSO 全局优化 + DNN（替代FDTD）
    ↓ 图像滤波 + 对称处理
  候选超表面模型

【验证阶段】
  FDTD 再验证 + 电场分析
    ↓
  5–8 μm 双功能（制冷+隐身）超表面
```

## 七、与相关工作的对比

### 7.1 与现有 ML 超表面设计方法对比

| 维度 | 传统 ML 方法 | 本文方法 |
|------|-------------|----------|
| 结构图案 | 简单几何图元（圆、方、十字等） | **Conway 细胞自动机生成的拟态复杂图案** |
| 材料库规模 | 单种或少量材料（1–5 种） | **71 种候选材料** |
| 材料编码 | 材料索引/物性参数 | **one-hot 编码，无序数偏差** |
| 图案编码 | 直接像素展开（高维冗余） | **AE 压缩至 256 维潜在空间** |
| 后处理 | 无 | **图像滤波 + 对称处理** |
| 设计映射 | 一对一（单目标单解） | **一对多（单目标多候选）** |

### 7.2 与随机像素图案的对比

- 相同填充率：Conway 图案光谱选择性更高、发射强度更强、寄生峰更少（图 S10）
- **原因**：Conway 规则赋予图案内在的空间相关性，使共振模式更集中、耦合更有效

### 7.3 与传统 MIM 结构对比

- 传统 MIM：几何参数有限（如圆盘直径、间距），设计空间相对简单
- 本文 MIM：引入 64×64 Conway 图案作为顶层，指数级扩展了结构设计空间

## 八、局限性与未来展望

### 8.1 局限性
- **仅数值验证**：本文仅进行了 FDTD 仿真验证，尚未有实验制备和测量结果
- **制备挑战**：尽管图像滤波后处理了边缘问题，复杂的 Conway 拟态图案仍对光刻精度提出更高要求
- **单一大气窗口**：当前设计聚焦于 5–8 μm 窗口，8–13 μm 窗口（另一重要辐射制冷波段）未涵盖

### 8.2 未来方向（作者提出）
- 将此设计范式拓展至更多功能的多光谱控制（如同时优化两个大气窗口）
- 利用更复杂的机器学习结构（如 Transformer）进一步提升预测精度
- 将训练好的模型用于指导实验样品制备和测试

- **与 s43074-025-00177-4（META 光源，PhotoniX 2025）对比**：
  - 两篇文章都在红外波段利用超表面（MIM 结构），但出发点不同：

  - 核心物理机制均涉及金属-介质-金属间的局域等离激元共振

- **与辐射制冷领域（研究方向）的关联**：
  - 本文直接针对 5–8 μm 辐射制冷窗口设计，与用户研究方向高度相关
  - Conway 图案生成 + 机器学习的设计流程可借鉴用于 100nm 以下超薄氟化物薄膜的光学设计探索

- **与 SUNDIAL 复现项目的关联**：
  - 两者都是"机器学习 + 光学/光谱设计"，但路径相反：
    - SUNDIAL：从测量光谱（Ψ/Δ）逆向推断材料参数（n, k, d）
    - 本文：从目标光谱（吸收率曲线）逆向推断结构参数（图案+材料+几何）

### 9.3 值得追踪的技术点

1. **Conway 图案库公开性**：文中 6000 种图案和 30000+ 数据集是否开源？（Data availability 仅指向 ESI，未提及代码库）
2. **中间聚合物层（PEI）选材**：PEI 的 5–8 μm 化学键吸收特性是本设计能成功的关键，其他具有类似特性的聚合物（如 PE、PMMA）能否替代？
3. **从 5–8 μm 扩展至 8–14 μm**：后者是更重要的大气窗口（对应人体/室温物体峰值辐射），需要更长波段的材料和更大周期结构
